# To-Do Web Application

## Live Demo
https://aravindkalva.github.io/SCT_WD_4/

A simple and responsive To-Do List web application built using HTML, CSS, and JavaScript.

This application helps users manage daily tasks by adding, editing, completing, and organizing tasks with date and time.

---

## Features

-  Add new tasks
-  Edit existing tasks
-  Delete tasks
-  Mark tasks as completed
-  Set date and time for tasks
-  Organize tasks using categories
-  Filter tasks:
  - All tasks
  - Completed tasks
  - Pending tasks
-  Data persistence using Local Storage
-  Responsive design for desktop and mobile devices

---

## Technologies Used

- HTML5
- CSS3
- JavaScript

---

## Application Features

### Add Tasks
Users can enter a task, select a category, set a date and time, and add it to the task list.

### Manage Tasks
Users can:
- Update task details
- Mark tasks as completed
- Remove unwanted tasks

### Task Storage
Tasks are saved using browser Local Storage, so data remains available after refreshing the page.
